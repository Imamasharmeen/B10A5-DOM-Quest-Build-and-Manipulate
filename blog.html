Important FAQ

What is Document Object Model(DOM)?

Answer is here

Document Object Model is the application programming interface for web documents. It defines the structure of a webpage, which can be dynamically accessed and manipulated by programs and scripts, hence altering the content, structure, and style of the document.

How do you select an element from the DOM?

Answer is here

To select an element in the DOM, JavaScript uses either of document.querySelector() or document.getElementById(). For Example: document.querySelector('selector') document.getElementById('id')

What is event delegation in the context of the DOM, and why is it useful?

Answer is here

Event delegation is a way to add just one event listener on some parent element to handle events occurring on its child elements, as a result it reduces the number of event listeners. Hence, this improves the performance and enables the handling of dynamically added elements.

How do you manipulate an element's attributes and styles using the DOM?

Answer is here

Changing the Attributes and Styles of an Element with DOM As was said before, in order to manipulate attributes and styles of an element with DOM, at first an element in the document must be selected. Having selected an element, any attribute can be changed by adding, removing, or updating values such as `src`, `href`, and `class`. One can manipulate CSS properties on elements directly through modification of the style object for styles. This will let one dynamically change the look and behavior of elements on the webpage.
